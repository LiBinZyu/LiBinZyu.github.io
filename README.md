# Personal Portfolio | 个人作品集

My personal portfolio website built with HTML, CSS, and Vanilla JavaScript.

## Features

- **Projects**: Filterable projects showcasing XR, 3D Art, AI, and Research.
- **Experiences**: Timeline showing education and professional work experience.
- **Blog**: Guide articles on Unity, OpenXR, and robot teleoperation.
- **Aesthetics**: Glassmorphism UI, light/dark mode, and bilingual support (EN/ZH).

## Development

Open `index.html` directly in the browser or serve it using a local server:

```bash
npm install -g live-server
live-server
```
